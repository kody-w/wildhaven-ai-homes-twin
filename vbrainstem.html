        Bundles every agent in your binder into a swarm package — a single JSON file containing each agent's source, the system soul, and a one-line install command. Drop the bundle on any machine running a Tier 2 endpoint (locally or in Azure) and you'll have a permanent, callable swarm at a known URL.
